# Z-Tech Technology Solutions 💻

**Z-Tech** is a modern, responsive e-commerce web application designed for a premium electronics retail business. This project was developed as a Final Year Diploma Project to demonstrate proficiency in core web technologies and user-centric design.



## 🚀 Key Features
- **Dynamic Product Grid:** Products are rendered dynamically from a JavaScript data structure.
- **Advanced Filtering:** Users can search for specific laptops or filter by category (Laptops vs. Accessories).
- **Smart Sorting:** Sort products by price (Low to High / High to Low).
- **Dark Mode Support:** A persistent theme-toggle that remembers user preference using LocalStorage.
- **Direct-to-WhatsApp Ordering:** Clicking "Buy" generates a pre-filled WhatsApp message for instant customer-to-seller communication.
- **Social Media Integration:** Built-in links to Facebook and YouTube for brand engagement.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure for SEO and accessibility.
- **CSS3:** Custom styling using CSS Variables and Flexbox/Grid for responsiveness.
- **JavaScript (ES6+):** Logic for filtering, sorting, and DOM manipulation.
- **GitHub Pages:** For live hosting and deployment.

## 📱 Mobile Responsive
The site is fully optimized for mobile devices, ensuring shopkeepers and customers can interact with the platform on the go.
